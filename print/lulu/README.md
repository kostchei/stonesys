# Lulu Print Build

Generated from `handbook/`.

## Outputs

- `interior.md` - assembled print source
- `interior.pdf` - Lulu-ready interior PDF, generated by `build_lulu_pdf.py`
- `lulu.config.json` - trim, margin, and typography settings
- `cover/` - placeholder directory for Lulu cover template/assets after final page count

## Build

```powershell
node handbook/tools/build-lulu-print.mjs
& 'C:\Users\Admin\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe' handbook/tools/build_lulu_pdf.py
```

The current target is a no-bleed US Trade 6 x 9 inch text interior. Upload the cover separately in Lulu after the final page count is confirmed.
