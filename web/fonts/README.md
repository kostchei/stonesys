# Fonts

Self-hosted, open-licensed old-printing serifs used across the app.

- **IM Fell English** (`imfell-*.woff2`) — Igino Marini's digitization of the
  17th-century John Fell types (Oxford University Press). Used for headers,
  titles, and flavor text for a crumbly, letterpress feel.
- **EB Garamond** (`ebgaramond-*.woff2`) — Georg Duffner & Octavio Pardo's
  revival of Claude Garamont's 16th-century types. Used for body and UI text
  for legibility.

Both are licensed under the SIL Open Font License 1.1 (see the `*-OFL.txt`
files). Files sourced from the Fontsource distribution.
